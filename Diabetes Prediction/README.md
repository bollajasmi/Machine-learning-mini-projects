# Diabetes Prediction using Decision Tree

## Description

This project uses a Decision Tree model to predict whether a person has diabetes based on medical data.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Dataset

* Input: Pregnancies, Glucose, BMI, Age, etc.
* Output:

  * 0 → No Diabetes
  * 1 → Diabetes

## Steps

1. Load dataset
2. Split data into training and testing
3. Train Decision Tree model
4. Predict results
5. Check accuracy
