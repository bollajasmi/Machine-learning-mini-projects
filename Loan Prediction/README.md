# Loan Prediction Project

## Description
This project is used to predict whether a loan will be approved or not based on customer details like income, education, loan amount, etc. It uses machine learning techniques.

## Dataset
The dataset includes the following details:
- Gender
- Married
- Education
- Applicant Income
- Loan Amount
- Credit History
- Loan Status (Yes/No)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Steps
1. Data Collection
2. Data Preprocessing (handling missing values and encoding)
3. Data Visualization
4. Model Training
5. Model Evaluation
6. Prediction

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Output
The model predicts loan status:
- Approved (Yes)
- Not Approved (No)

## Conclusion
This project helps in predicting loan approval and supports decision-making in banks using machine learning.
